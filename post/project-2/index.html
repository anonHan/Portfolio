Rotten Tomatoes Web Scrapper

By Sahil

June 15, 2022

Introduction:

We all data enthusiasts know the one of the famous saying that Data is the new Oil. But there comes a big question in my mind every time I come across with it that:

  • How would you get this oil?
  • How would you mine it?
  • How much amount do we need to pay?
  • Is there any special machinary used to extract this data?

Objective:

With all these questions in my mind, I came up with an idea to build a web scrapper to scrape the reviews. The next question that I had is that which website should I start scrapping? After doing some research I decided to scrape the world-famous website Rotten Tomatoes.

  • Why Rotten Toamtoes?

    People had been using Rotten Tomatoes to find movie reviews since it launched in 2000 and has a huge user base, which means more data/reviews to scrape.

    Another reason for choosing Rotten Tomatoes is that I have already seen a lot of people using IMDB dataset. So thought of trying somethng new.

Solution:

Build a simple tool to scrape all the reviews of a particular movie using the Python and Selenium. This tool can gather all the reviews given by the different users to any movie on the website (as long as the website contains the reviews of the provided movie) and saves it in the CSV format in the local system.
